## 課題
本課題では、全国の県庁所在地の消費データを読み込み、ソートするプログラムを作成する。

1. 使用するデータ  
全国の県庁所在地の消費データは、consumption.csvに保存されている。consumption.csvには、csv形式で、以下のデータが含まれる。
- ID
- 市の眺め
- 世帯人員
- 食料合計
- 穀類
- 魚介類
- 肉類
- 乳卵類
- 野菜類
- 果物
- 調味料
- 菓子類
- 調理料理
- 飲料
- 酒類
- 外食

県庁所在地の消費データは、下記構造体の配列に読み込まれる。

```
typedef struct {
    int id;
    char name[19];  //  市の名前、UTF-8対応
    float members;  //  世帯人員
    int total;      //  食料合計
    int grain;      //  穀類
    int seafood;    //  魚介類
    int meat;       //  肉類
    int milk;       //  乳卵類
    int vegetable;  //  野菜類
    int fruit;      //  果物
    int seasoning;  //  調味料
    int snack;      //  菓子類
    int cocking;    //  調理料理
    int drink;      //  飲料
    int liquor;     //  酒類
    int eatout;     //  外食
} City;
```

## ソースコードの説明
バブルソート
①入力回数を初期化する。
②数値列の先頭から順番に右隣と値を比較する。
③左のほうが大きい場合、右隣と入替、入替回数を増やす。
④そうでない場合はそのまま。
⑤数値列の最後の比較場所まで、②~④の処理を繰り返す。
⑥入替回数が０であれば終了。０がでない場合、①から繰り返す。

クイックソート
①要素が1以下であれば終了
②探索用indexを左端と右端にする
③左端の値をPivotにする
④左から順番にPivotより大きな値を探す
⑤右から順番にPivot以下の値を探す
⑥iとjの位置が同じか、jの値がiよりも小さければ以下の処理を行う
    ⑴Pivotの値とjの値を入れ替える
    ⑵jの位置の左側と右側に配列を分割し、①の処理を再帰的に呼び出す
⑦iがjよりも小さければ、iの場所の値と、jの場所の値を入れ替え、④から繰り返す。

マージソート
①要素が1以下であれば終了
②真ん中を計算する
③左側をマージソートする
④右側をマージソートする
⑤マージする
